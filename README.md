**Key Abbreviations:**
- **Neurotransmitters:** NE = Norepinephrine, DA = Dopamine, ACh = Acetylcholine, 5-HT = Serotonin, Glu = Glutamate
- **Brain Regions:** PFC = Prefrontal Cortex, dlPFC = Dorsolateral PFC, vmPFC = Ventromedial PFC, ACC = Anterior Cingulate Cortex, IFG = Inferior Frontal Gyrus, OFC = Orbitofrontal Cortex, FEF = Frontal Eye Fields, TPJ = Temporoparietal Junction, MTL = Medial Temporal Lobe, MS/DB = Medial Septum/Diagonal Band
- **Nuclei:** LC = Locus Coeruleus, VTA = Ventral Tegmental Area
- **Molecular:** GABA = Gamma-aminobutyric acid, CREB = cAMP response element-binding protein, BDNF = Brain-derived neurotrophic factor, TrkB = Tropomyosin receptor kinase B, LTP = Long-term potentiation, NMDA = N-methyl-D-aspartate, mGluR = Metabotropic glutamate receptor

## Family Characteristics Summary

**Attention Family**: Foundational functions that gate information processing across all other cognitive domains.

**Memory Family**: Systems for encoding, storing, and retrieving information across different timescales and content types.

**Executive Function Family**: Higher-order control processes that regulate and coordinate other cognitive functions.

**Processing Speed Family**: The efficiency and speed of basic cognitive operations.

**Language Family**: Specialized systems for linguistic processing, comprehension, and expression.

**Visuospatial Family**: Processing of spatial relationships, visual patterns, and visuomotor integration.

**Reasoning & Abstract Thinking Family**: Higher-order cognitive processes for logic, abstraction, and complex problem-solving.

**Learning & Adaptation Family**: Mechanisms for acquiring new skills, knowledge, and adapting to environmental changes.
